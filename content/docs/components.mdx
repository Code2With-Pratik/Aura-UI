## Navbar

A horizontal, full-width nav with a wide search trigger. Clicking the search field — or pressing `⌘K` / `⌃K` anywhere — opens the **Spotlight** modal.

## Spotlight

A keyboard-driven command palette inspired by macOS Spotlight. Filter by typing, navigate with `↑` `↓`, activate with `Enter`, dismiss with `Esc`.

## Hero

Radial glow + blinking-status badge + Instrument Serif display headline that scales with `clamp()`.

## Window Mockup

Functional macOS window: traffic-light controls (yellow click triggers a genie minimize), sidebar with grouped items, and a 3-column preview grid of tiny live components.

## Bento Grid

12-column grid with 4 / 5 / 7 spans, each card carrying a unique data-viz visual and a hover-lift on the same `--ease-aura` curve.

## Marquee

A CSS-only horizontal ticker for changelog or status updates, with edge-fade masks on either side.
